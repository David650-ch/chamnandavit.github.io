Email Forwarding Setup Guide

Learn how to automatically forward emails using Thunderbird, Outlook, and Gmail

Mozilla Thunderbird
  1. Open Thunderbird and select your account from the left panel.
  2. Click Tools > Message Filters.
  3. Click "New" to create a new filter.
  4. Give the filter a name like Forward to personal.
  5. Set the filter condition (e.g., "Match all messages").
  6. In the actions, choose Forward Message To and enter the forwarding address.
  7. Click OK and ensure the filter is enabled.
Tip: Filters only apply to messages as they arrive. They wonâ€™t retroactively forward old emails.
Microsoft Outlook
  1. Go to File > Manage Rules & Alerts.
  2. Click "New Rule" under the Email Rules tab.
  3. Select "Apply rule on messages I receive" and click Next.
  4. Choose any conditions you want (or none to apply to all emails).
  5. In actions, check "forward it to people or public group".
  6. Select the forwarding email address and finish the rule.
Note: Outlook desktop rules only run when Outlook is open, unless configured on the server side (Exchange).
Google Mail (Gmail)
  1. Click the gear icon and select See all settings.
  2. Go to the Forwarding and POP/IMAP tab.
  3. Click "Add a forwarding address" and enter the desired email.
  4. Check your other inbox and confirm the verification link.
  5. Back in Gmail settings, choose "Forward a copy of incoming mail to...".
  6. Decide if Gmail should keep a copy, archive, or delete it.
  7. Click Save Changes.
Tip: You can also set filters to forward only specific types of emails.
